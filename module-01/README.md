

## План заняття:

- Вебтехнології: HTML, CSS, JavaScript
- Інструменти розробника: Chrome DevTools
- Скелет HTML-документа: `doctype`, `html`, `head`, `body`
- Метаінформація документа: `title`, `meta`
- Теги та атрибути: `h1...h6`, `p`, `a`, `img`
- Парні та одиночні теги
- Коментарі
- Списки: `ul`, `ol`, `li`
- Семантична розмітка: `header`, `footer`, `main`, `section`, `nav`
- Блокові та рядкові елементи
- Вкладеність тегів
- Посилання: `href`, `target`, `tel:`, `mailto:`, `download`
- Тег `address`
- Тег `button`. Різниця між кнопкою та посиланням
- Ієрархія заголовків
- Оформлення коду: відступи та форматування


- Структурна розмітка
- Алгоритм створення сторінки
