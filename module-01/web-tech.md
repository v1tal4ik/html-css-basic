# 01. Вебтехнології

## Що таке вебтехнології

Вебтехнології — це набір технологій для створення сайтів та вебзастосунків.

Основні вебтехнології:

- HTML — структура сторінки
- CSS — зовнішній вигляд
- JavaScript — логіка та взаємодія

## Як працює сайт

1. Користувач відкриває сайт.
2. Браузер отримує файли від сервера.
3. HTML створює структуру сторінки.
4. CSS додає стилі.
5. JavaScript додає інтерактивність.

## Приклад

Сайт можна порівняти з будинком:

- HTML — каркас
- CSS — ремонт і дизайн
- JavaScript — електрика та механізми

# HTML, CSS, JavaScript

## HTML

HTML (HyperText Markup Language) — мова розмітки для створення структури сторінки.

```html
<h1>Заголовок</h1>
<p>Абзац тексту</p>
```

## CSS

CSS (Cascading Style Sheets) — мова стилів для оформлення сторінки.

```css
h1 {
	color: blue;
}
```

## JavaScript

JavaScript — мова програмування для створення логіки та взаємодії.

```javascript
alert("Hello World");
```

## Важливо

HTML → структура

CSS → дизайн

JavaScript → поведінка

# Інструменти розробника (DevTools)

## Що таке DevTools

DevTools — вбудовані інструменти браузера для аналізу та налагодження сайтів.

## Як відкрити

F12 або Ctrl + Shift + I

## Основні вкладки

### Elements

Перегляд HTML.

### Styles

Перегляд CSS.

### Console

Виконання JavaScript.

### Network

Перегляд файлів, які завантажує сторінка.

# Що таке HTML

## HTML

HTML — стандартна мова розмітки вебсторінок.

HTML не є мовою програмування.

HTML описує структуру документа за допомогою тегів.

## Що таке тег

Тег — це команда в кутових дужках.

```html
<h1></h1>
<p></p>
<a></a>
<img>
```

## Структура HTML документа

```html
<!DOCTYPE html>
<html>
<head>
	<title>Мій сайт</title>
</head>
<body>
	Контент сторінки
</body>
</html>
```

## Основні теги

Заголовок:

```html
<h1>Заголовок</h1>
```

Абзац:

```html
<p>Текст</p>
```

Посилання:

```html
<a href="#">Посилання</a>
```

Зображення:

```html
<img src="image.jpg" alt="Image">
```

# Висновки

- Вебтехнології використовуються для створення сайтів.
- HTML створює структуру сторінки.
- CSS відповідає за зовнішній вигляд.
- JavaScript додає логіку та взаємодію.
- DevTools допомагають аналізувати та знаходити помилки.
- HTML є основою будь-якої вебсторінки.
