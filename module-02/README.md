## План заняття:

- Підключення стилів (inline, internal, external)
- Колір (color, background-color, формати, прозорість)
- Селектори (тег, #id, .class, нащадок, дочірній, псевдокласи)
- Каскад стилів (специфічність, !important, успадкування, currentColor)
- Шрифт (font-family, size, weight, style, Google Fonts, @font-face)
- Оформлення тексту (decoration, transform, align, spacing, shadow, span)
- Глобальні стилі. Нормалізація (modern-normalize)
- CSS змінні (оголошення, використання)
- Композиція класів (модифікатори, комбінація селекторів)
- Автоформатування у VS Code (Prettier + formatOnSave)

# Алгоритм додавання CSS стилів

## Процес стилізації сторінки

Процес стилізації складається з кількох етапів:

1. Підключення шрифтів — Google Fonts.
2. Підключення нормалізації — modern-normalize.
3. Створення файлу стилів — css/styles.css.
4. Написання базових стилів — скидання та загальні правила.
5. Додавання класів до HTML — іменування за BEM-подібною схемою.
6. Стилізація секцій — від header до footer.

## Важливо

- Спочатку підключаємо зовнішні ресурси, потім пишемо власні стилі.
- Класи додаємо поступово, секція за секцією.
- Після кожного кроку перевіряємо результат у браузері.
